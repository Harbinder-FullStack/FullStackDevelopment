🌐 Node.js Execution & API Access

This lecture explains how we can execute Node.js code directly, through npm, interactively, or as APIs accessed from browsers, frontend apps, and tools.

1. Direct Execution

Run your Node.js file directly in the terminal.

node app.js

2. npm Scripts

Define scripts in package.json and execute with npm.

"scripts": {
  "start": "node app.js",
  "dev": "nodemon app.js"
}
# Run with:
npm start
npm run dev

3. REPL (Interactive Mode)

Start Node.js REPL (Read–Eval–Print Loop) and execute JS line by line.

node
> console.log("Hello from REPL")
Hello from REPL

4. Accessing Node.js APIs

When Node.js runs as a server (e.g., Express), it provides APIs that can be accessed in multiple ways.

┌─────────────────────────────┐ │ Node.js Server (Express) │ │ app.get('/api/hello') {...}│ └──────────────┬──────────────┘ │ ┌─────────────────────┼─────────────────────┐ │ │ │ Browser (URL) React Frontend API Tools http://localhost:5000 fetch("/api/hello") Postman / curl 
// Example Express API
const express = require('express');
const app = express();
app.get('/api/hello', (req,res) => res.json({msg: "Hello API!"}));
app.listen(5000, () => console.log("Server running on 5000"));

5. CLI Tool with Arguments

Node.js can accept command-line arguments using process.argv.

# Run:
node app.js John 25

// app.js
const args = process.argv;
console.log("Name:", args[2]);
console.log("Age:", args[3]);

6. Background Worker / Scheduled Jobs

Node.js scripts can run at intervals using cron or libraries like node-cron.

const cron = require('node-cron');

// Run every minute
cron.schedule('* * * * *', () => {
  console.log("Task running every minute");
});
